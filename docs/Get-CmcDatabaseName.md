# Get-CmcDatabaseName

## SYNOPSIS
Get Commence database name.

## SYNTAX

```
Get-CmcDatabaseName [<CommonParameters>]
```

## DESCRIPTION
Gets the name of the running Commence database.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-CmcDatabaseName
```

Gets the name of the running Commence database.
