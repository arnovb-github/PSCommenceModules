# Get-CmcCategories

## SYNOPSIS
Get category names from Commence

## SYNTAX

```
Get-CmcCategories [<CommonParameters>]
```

## DESCRIPTION
Returns the category names of the currently active Commence database.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-CmcCategories
```

Returns the category names of the currently active Commence database.
